Daftar Isi Apa itu Tmux? Membuat Session Baru Membuka Tmux Session Apa itu Tmux? Tmux adalah aplikasi yang memungkinkan kita membuat session tersendiri di dalam terminal shell. Session tersebut bisa kita biarkan berjalan walaupun terminal sudah diclose. Keuntungan memakai session tmux salah satunya ketika kita mengakses shell secara remote menggunakan ssh, kita bisa mengakses session tmux yang sedang aktif. Untuk menginstall tmux pada distro linux anda, silahkan melihat referensi pada repository resmi tmux
