Contoh Perintah 1 ln -s /home/joenmarz/folder1/file_asli.txt file.txt Setelah menjalankan perintah ini kita bisa mengakses /home/joenmarz/folder1/file_asli.txt dengan file.txt Jika terdapat perubahan pada file.txt maka akan berpengaruh juga pada file aslinya. Cara Melepas Symlink Sebelum melepas (remove) symlink, kita perlu mengkonfirmasi bahwa file ataupun folder tersebut benar merupakan symlink. 1 ls -l <path-ke-symlink> Menjalankan perintah ini di terminal akan menampilkan properties dari file.
