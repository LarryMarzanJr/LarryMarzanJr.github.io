### Ethereum (ETH)
Red Panda Mining's setup:
```
Power Limit: 100
Temp Limit: 83
Core Clock: -502
Memory Clock: +1300
Fan Speed: 60
```

My custom setting:
```
Power Limit: 76
Temp Limit: 73
Core Clock: -502
Memory Clock: +1300
Fan Speed: 60
```

### Ergo (ERG)
Red Panda Mining's setup:
```
Power Limit: 45
Temp Limit: 68
Core Clock: +0
Memory Clock: +0
Fan Speed: 80
```
Or reset afterburner to stock, on t-rex miner, add `--lock-cclock 1520` and run as administrator.
