# Proses Instalasi

Buat project baru Jekyll, misalnya dengan nama `blog_baru`.
```
jekyll new blog_baru
```

Masuk ke dalam direktori project:
```
cd blog_baru
```

Kemudian jalankan server lokal Jekyll:
```
bundle exec jekyll serve
```
